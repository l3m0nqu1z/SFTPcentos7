# SFTPcentos7
Centos7(MTS) image's SFTP builder
Autobuilder sftp based on Centos 7 image.

Script's steps
-updating system
-instaling fail2ban
-extending disk
-adding users >> STDOUT
PROFIT

If you want to create more than one user just launch script with argument=number users, example:
./sftpcetos7 5 ---> create 5 users
